Bloatfish is a command-line tool that audits your web projects against **GreenIT** and **RGESN** (French Eco-Design Reference Framework) principles. It combines a **Go-based CLI** with an **ESLint plugin** to provide comprehensive environmental impact analysis.

## Features

### Comprehensive Analysis

- **Dependencies** ‚Äî Weight tracking, unused packages detection, framework exclusions
- **Images** ‚Äî Format optimization, size analysis, duplicate detection
- **Fonts** ‚Äî Weight monitoring, format recommendations, variant optimization
- **JavaScript & CSS** ‚Äî Bundle size analysis and optimization hints
- **Videos** ‚Äî Size tracking and autoplay detection
- **Routes & Pages** ‚Äî Structure analysis and lazy-loading recommendations
- **Code Quality** ‚Äî ESLint integration with eco-design rules

### Smart Reporting

- **EcoScore** (0‚Äì100) ‚Äî Single metric to track your project's environmental impact
- **JSON Report** ‚Äî Machine-readable data for CI/CD integration
- **HTML Dashboard** ‚Äî Visual, Lighthouse-style report with detailed insights
- **SVG Badge** ‚Äî Display your EcoScore in your README
